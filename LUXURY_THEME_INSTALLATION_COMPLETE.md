## ğŸ“ Quick Customization

### Change Primary Color
Edit `luxury-pos.css`:
```css
:root {
    --primary-gold: #YOUR_HEX_COLOR;
}
```

### Change Accent Colors
```css
:root {
    --success-color: #YOUR_GREEN;
    --danger-color: #YOUR_RED;

}
```

```css
:root {

}
```

```css
}
```
